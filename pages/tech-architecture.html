00 — System Overview

五大模块如何咬合

自下而上看：知识中台提供“记忆”，技能工具层提供“手脚”，编排引擎作为“大脑”调度二者， Agent Builder 让用户在上方搭建专属分身，数据飞轮作为横切面贯穿全程，持续把执行结果反哺为新的能力。 五层共同构成“感知 → 决策 → 执行 → 反馈 → 进化”的完整闭环。

01 — Core Brain

Agent Orchestration Engine

底座的“大脑”。解决“如何把 Harness 能力组装起来”的问题。

① Multi-Agent Framework

多 Agent 协作框架

基于 Mastra（TypeScript 原生 Agent 框架）搭建多智能体协作体系。 支持 DAG 有向无环图工作流编排，让“选题 Agent”“文案 Agent”“分发 Agent”像流水线一样协同工作； 并通过其原生的 Agent Network 与 Workflow 能力，支持顺序、并行、循环等多种协同模式。

② Dynamic Planner

动态任务规划

底层需要一个强大的 Planner，能够根据用户的自然语言指令—— 比如“帮我策划一个五一假期的景区促销”—— 自动拆解任务，并调用底层的 Skills 和 Tools。

③ State Management

状态管理

文旅营销是长链路任务，引擎必须能精准记忆上下文状态， 确保在多轮对话和复杂任务执行中不丢失信息。

02 — Knowledge Infrastructure

RAG 2.0 与向量知识中台

底座的“记忆库”。区分官方知识与用户私有认知的关键。

① Multi-Tenant Isolation

多租户知识隔离

在向量数据库（Milvus / Qdrant）层面实现严格的 Namespace 命名空间隔离。 官方分身调用行业通用库，自定义分身调用“行业库 + 自有私有库”。

② Hybrid Retrieval

混合检索策略

文旅行业有大量专有名词（特定景区 IP、非遗名称），纯向量检索易出错。 采用 向量检索 + 关键词检索（BM25）+ 知识图谱 的混合模式，提高召回准确率。

③ Live Update Pipeline

知识动态更新管道

文旅热点变化极快。需搭建自动化 ETL 管道， 实时抓取全网文旅热点、平台算法规则变化，自动清洗并向量化注入知识库。

03 — Capability Layer

Skill / Tools 注册与网关

底座的“手脚”。实现能力的原子化封装。

① Standard Protocol

标准化 Skill 协议

定义一套标准的 Skill 描述协议（基于 JSON Schema 或 OpenAPI）。 无论官方“脚本生成 Skill”，还是用户上传的“私域客服 Skill”，都必须遵循该协议才能被 Agent 调用。

② API Gateway

API 聚合网关

统一管理对抖音、小红书、微信等第三方平台 API 的调用。 做好限流、鉴权、异常重试机制， 避免因某个平台接口波动导致整个 Agent 瘫痪。

③ Sandbox Execution

沙箱执行环境

当用户自定义分身调用复杂代码或工具时，必须在隔离的沙箱环境中执行， 防止恶意代码或死循环拖垮整个底座平台。

04 — User DIY Layer

低代码 / 无代码 Agent Builder

实现“用户 DIY 专属分身”的前端交互核心。

① Visual Workflow Editor

可视化工作流编辑器

提供类似 Coze / Dify 的拖拽式界面。 用户可通过连线方式，把底座的 RAG、Skill、Tools 像搭积木一样组合起来。

② Prompt IDE

提示词工程辅助

可视化的 Prompt 调试界面，帮助用户注入自己的“认知”。 提供 Few-shot 少样本学习模板—— 用户上传自己品牌的优秀文案作为范例，Agent 就能快速学会这种调性。

③ Simulation Sandbox

仿真测试环境

在用户发布自定义 Agent 前，提供仿真沙盒， 用历史数据跑一遍，评估其生成内容的质量和合规性。

05 — Evolution Loop

Data Flywheel 数据飞轮

确保“官方分身越来越聪明，底座越来越强”的闭环。

① Full-Chain Tracing

全链路埋点与追踪

记录 Agent 的每一次思考链（CoT）、 每一次工具调用、以及最终的业务结果（视频播放量、核销率）。

② RLHF Interface

RLHF 人类反馈接口

在官方分身和自定义分身后台，提供便捷的点赞 / 点踩 / 修改接口。 用户反馈自动回流，用于微调底层 LLM 或优化 RAG 检索权重。

③ A/B Testing

A/B Testing 框架

支持对不同 Agent 策略进行灰度测试，自动筛选出转化率最高的营销 SOP， 并将其固化为新的官方 Skill。
