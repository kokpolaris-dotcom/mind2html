01 — Industry Re-map

重新定义行业参与者

从“谁能解决营销问题”的视角，文旅行业存在三类角色，各自卡在不同环节。

01 上游 · 中游

资源方 & 服务商

有产品，但无营销能力

核心痛点

景区、酒店、体验服务商多是“资源型 / 服务密集型”企业。擅长做服务、搞基建，面对抖音、小红书等新媒体极度缺乏专业营销域能力——不懂抓热点、不懂写爆款脚本、不懂流量转化。

业务解法

通过代运营服务或数字化产品弥补内容营销、流量获取与转化闭环的短板，把“资源”转化为“客流”。

02 人力驱动

传统代运营团队

解决 0→1 冷启动，但难规模化

核心痛点

人力代运营是非标、重资产、难规模化。金牌运营一年能服务的客户有限，人员流动导致服务质量波动，边际成本极高。

业务解法

提供定制化专业策略，但难以覆盖全行业海量商家的普惠需求。

03 Wanpix.ai

营销域能力 Agent 化

专家级能力的规模化复制

核心痛点

行业需要“懂文旅营销的专家”，但专家太贵且太少。纯工具不懂业务，纯人力跑不快。

业务解法

把“金牌运营”的人脑经验，转化为 AI Agent 的执行逻辑——永不离职、24h 在线、拥有全行业顶尖营销经验。

04 — Two-Layer Architecture

双层级产品架构

把“能力”变成可组装的“乐高积木”——商业天花板从 SaaS 工具跃升到 PaaS 生态平台。

L1 Base Platform

底座平台

能力原子化 · 行业基建

Harness 技术（RAG / Skill / Tools）本质是把代运营团队积累的“营销域核心能力”进行结构化拆解与原子化封装——文旅营销领域的“操作系统”。

RAG

行业知识库

沉淀爆款文案库、平台算法规则、历史转化数据等“隐性知识”。

Skill

标准化技能

封装具体营销动作：热点捕捉、脚本生成、团购套餐设计。

Tools

执行接口

打通抖音、小红书等平台 API，解决“最后一公里”分发。

L2 Official Avatar

官方专家分身

标准化交付 · 标杆示范

利用底座 Harness 能力，组装出“开箱即用”的标准化产品。解决行业 80% 的通用需求——是底座的旗舰应用与样板间。

面向中小商家——无需懂底层逻辑，直接带来客流的“超级数字员工”。

既是直接营收来源，又向市场证明底层能力的强大。

L3 Hidden Value

用户自定义分身

生态化 · 长尾赋能

允许大型文旅集团、MCN、资深代运营调用底座 Harness 能力，注入私有认知（品牌调性、私域逻辑、客群洞察），构建专属 Agent。

长尾

个性化需求

文旅是非标行业，头部大客户有极个性化营销逻辑，官方分身无法完全覆盖。

粘性

Lock-in 效应

大客户用底座 RAG 与 Tools 搭建专属工作流后，迁移成本极高。

飞轮

反哺底座

自定义产生的新玩法、新认知，可被提炼回底座，形成生态飞轮。
